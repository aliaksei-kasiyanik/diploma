Explicit Runge-Kutta Method Input Scheme
Схема ввода неявного метода Рунге-Кутта
----------------------------------------

    s  //the number of stages

    a_11 ... a_1s
     ... ... ...
    a_s1 ... a_ss

    b_1 ... b_s

    c_1 ... c_s

----------------------------------------
Ex.:

2

0.416666666666666 -0.08333333333333333
0.75 0.25

0.75 0.25

0.3333333333333333 1
